# Form UKT
Deskripsi Aplikasi

Aplikasi berbasis Desktop yang berisi formulir untuk mengajukan jenis Uang Kuliah Tunggal (UKT) yang berlaku di Universitas Sumatera Utara menggunakan bahasa pemrogramman Visual Basic Net.
